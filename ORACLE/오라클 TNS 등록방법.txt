1.InstantClient 설치 여부 확인 및 listener.org 변경



oracle instantClient 설치

https://www.oracle.com/downloads/#category-database



 C:\app\[사용자명]\product\[오라클버전]\home\[xxxxx]\network\admin\listener.org 

를 확인하여 해당 파일에 현재 접속하는 서버의 IP주소가 제대로 입력되어있는지 확인

(LocalHost 로 입력된경우, 외부 접속 불가)

LISTENER =
  (DESCRIPTION_LIST =
    (DESCRIPTION =
      (ADDRESS = (PROTOCOL = TCP)(HOST = [ip주소])(PORT = 1521))
      (ADDRESS = (PROTOCOL = IPC)(KEY = EXTPROC1521))
    )
  )


2. 시스템환경변수 ORACLE_HOME, TNS_ADMIN 추가 및 PATH 에 

ORACLE_HOME과 동일하게 추가

TNS_ADMIN : InstantClient 설치 위치 

ORACLE_HOME, PATH : C:\app\[사용자명]\product\[오라클버전]\dbhome[SID명] 

* C:\app\[사용자명]\product\[오라클버전]\home 과 혼동하지 않기



*환경변수가 잘못되었을때 발생가능한 오류

- ORACLE_HOME : CMD - SQLPLUS 자체가 실행이 불가능

- TNS_ADMIN : 추후 TNS 작동을위해 TNSPING을 하는경우 에러 발생



3.tnsname.ora 및 sqlnet.ora 작성



*C:\app\[사용자명]\product\[오라클버전]\home\network\admin\sqlnet.ora

SQLNET.AUTHENTICATION_SERVICES= (NTS)
NAMES.DIRECTORY_PATH= (TNSNAMES, ONAMES, HOSTNAME)
#기존파일에서 EZCONNECT로 되어있던것을 변경
*[InstantClient 설치 위치] \tnsname.ora 

: 다음과 같은 형태로 TNS목록에 추가할 DB의 정보를 작성 

(YAML파일 처럼 공백을 잘 맞추어서 입력 필요 > 탭키 사용 시 에러발생)

[ALIAS명] =
  (DESCRIPTION =
    (ADDRESS = (PROTOCOL = TCP)(HOST = [ip명])(PORT = [포트]))
    (CONNECT_DATA =     
      (SERVICE_NAME = [SERVICE_NAME 입력])
      (SID = [SID 입력])
    )
  )
Service Name = 여러개의 인스턴스를 모아 하나의 서버 혹은 시스템을 구성한것

SID = DB 하나의 인스턴스



*확인 쿼리문 

sysdba 로 접속 후

Service_NAME : SELECT NAME, DB_UNIQUE_NAME FROM v$database;

SID : SELECT instance FROM v$thread;
